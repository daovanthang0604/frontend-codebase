# CLAUDE.md — dashboard (Frontend)

App-specific notes for the `dashboard` starter app (`apps/dashboard`). This is a
frontend-only template extracted from a larger product; the backend, its API
client, and the auth vendor were removed. See the root `README.md` for what is
stubbed and how to wire a real backend.

---

## Tech Stack

- **React + TypeScript** — single-page app (dev on port 3000).
- **TanStack Router** — typed file-based routes under `src/routes`; `routeTree.gen.ts` is generated by the Vite plugin (do not edit by hand). Search params validated with Zod.
- **TanStack Query** — server state via `useQuery` / `useMutation`; the shared axios instance and query wiring live in `src/services/`.
- **Tailwind CSS v4** — via PostCSS (`@workspace/theme/postcss.config`); design tokens come from `@workspace/theme`.
- **Shared packages** — `@workspace/ui` (component library, consumed from source), `@workspace/theme` (theme/tokens, consumed from source), `@workspace/shared` (utilities).
- **i18n** — `react-i18next`; `useTranslation` with the page name as the namespace; catalogs in `public/locales`.
- **Auth** — a **stub** boundary (`src/services/auth`): `signIn()`/`signOut()` mint/clear a fake token; `useAuthSession()` reports auth from token presence. Replace with a real provider.

---

## Code Conventions

1. **Reuse `@workspace/ui` first** — shared primitives live in `packages/ui/src` (imported as `@workspace/ui/components/*`). Reuse before building custom UI; many are named exports inside a shared file (e.g. `Input.tsx` also exports `TextArea`/`SearchInput`), so grep the exports, not just filenames.
2. **Wrap page content in `<PageShell>`** — it owns the outer gutter, centering, max width, and the title/description header. Don't hand-roll `h1` + `mx-auto … p-5`.
3. **`services/` is API and data-fetching only** — no business logic or feature constants.
4. **Use `@/` path aliases** — avoid relative imports more than 2 levels deep.
5. **Prefer precise types** — use `unknown` + type guards for untrusted values; `any` is allowed when it is the simplest honest fit.
6. **Use `&nbsp;` for non-breaking spaces in JSX**, not `{" "}`.
